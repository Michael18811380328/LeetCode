# LeetCode
中文乐扣算法练习 leetcode 1000道题 每天做5道，需要200天完成

https://leetcode-cn.com

学习步骤：

1. 熟练掌握数据结构和算法
2. 根据知识点刷题（针对训练）
3. 遗忘曲线复习
4. 总结题目到算法

一道题目使用尽可能多方法写出来：简单暴力方法；适合大部分情况的方法（5%的特殊情况不考虑）；事件复杂度最简单的做法；学习更好的算法。
