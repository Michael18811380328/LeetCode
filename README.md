# LeetCode 刷题记录

## 简介

乐扣算法刷题，网站链接 https://leetcode-cn.com。

注意：学习目的不是独创算法，而是学会已有的算法，掌握标准算法。所以每一个题目要看最佳的算法，然后学会这个算法，并不是随便解答出来。

学习步骤：

1. 熟练掌握数据结构和算法
2. 根据知识点刷题（针对训练）
3. 遗忘曲线复习
4. 总结题目到算法
